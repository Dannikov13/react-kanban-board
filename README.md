# Kanban Task Manager

📋 A responsive Kanban-style task manager built with React and TypeScript.

> **Status:** 🛠️ Project is currently in active development.

## 🎯 Project Goals & Features (Planned)
- [ ] Interactive Drag-and-Drop functionality
- [ ] Task and column management (Add/Edit/Delete)
- [ ] Data persistence via LocalStorage
- [ ] Responsive design for mobile and desktop

## 🛠️ Tech Stack
- React
- TypeScript
- Tailwind CSS
- Vite
